Fundamentos cinéticos

PulseGrowth usa el modelo de crecimiento exponencial para estimar la tasa específica de crecimiento (μ) entre dos puntos de muestreo y derivar el tiempo de duplicación.
X = X₀ · e^(μ · t)
μ = ln(X₁ / X₀) / Δt
td = ln(2) / μ
Para normalizar consumo de metabolitos, se estima IVCD asumiendo crecimiento exponencial en el intervalo. Esto mejora la interpretación de q frente a usar solo X inicial o final.
IVCD = ((X₁ − X₀) / ln(X₁ / X₀)) · Δt

Lógica de qGlc / qGln (corregida)

Para evitar inconsistencias entre entradas en g/L y mM, la app convierte internamente concentraciones de glucosa y glutamina a mM usando peso molecular (MW) y reporta siempre q en pmol/cell/day.
  • Glucosa: MW = 180.156 g/mol
  • Glutamina: MW = 146.145 g/mol
  • Convención de signo: q positivo = consumo; q negativo = acumulación / producción neta
  • Comparabilidad: ahora el resultado es comparable aunque ingreses g/L o mM
También se muestra el equivalente en pg/cell/day como referencia intuitiva de masa.
